# Securing Microsoft ADCS 

This documentation describes how to secure the private keys used by **Microsoft PKI Active Directory Certificate Services (AD CS)** (root, enterprise, subordinate CA) with Securosys Primus HSM or [CloudHSM](/cloudhsm/overview/) services.

:::tip New
Migrate from another key store to Primus HSM and reconfigure the AD CS server instance.
:::

For more information, please visit the [Microsoft ADCS section](../../ms-pki-adcs/overview).
